Click & Conquer: Dragon's Gem Quest

  • Tech Stack: Python, Pygame
  • Github URL: Project Link

Participate in a cooperative journey created for Python Game Jam #4, themed "The END?". This 2D adventure challenges players to work together to save the world from impending doom, marked by meteor showers and escalating threats.

Developed using the Pygame library in Python, this project was a meaningful learning experience for me. The collaboration with a friend and fellow developer made the process enjoyable and educational.

While I had the pleasure of contributing to various aspects of the game, my main responsibilities included:

  • Game Logic: I helped in shaping the basic rules and structure, aiming to create a coherent player experience.
  • Player Movement: My efforts were directed towards enabling smooth navigation for the players within the game environment.
  • Assets and Animation: I worked on integrating the assets and give life to the animations to increase the overall experience and player feel.
